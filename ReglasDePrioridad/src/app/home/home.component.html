REGLAS DE PRIORIDAD




Reglas de prioridad es una aplicación web encargada de resolver problemas de Scheduling. Para ello, se implementará varias de las heurísticas más populares. Esta aplicación proporcionará una interfaz que permita la ejecución de dichas heurísticas, con la posibilidad de ver los resultados de forma gráfica y descriptiva, permitiendo una comparación entre los comportamientos de las heurísticas elegidas. También almacena los datos de ejecuciones previas, teniendo la posibilidad de reutilizar los datos de entrada correspondientes a otras planificaciones o simplemente volver a visualizar los resultados anteriormente generados.
